# PDF RAG Chatbot using Groq + LangChain

An intelligent Retrieval-Augmented Generation (RAG) chatbot built using **LangChain**, **Groq LLM**, **Chroma Vector Database**, and **Streamlit**.

The chatbot reads a PDF file, converts it into embeddings, stores them in a vector database, and answers user questions using context-aware LLM responses.

---

## Features

- PDF document ingestion
- Automatic text chunking
- Semantic search using vector embeddings
- Context-aware answers using Groq LLM
- Interactive Streamlit chat UI
- Secure API key management using `.env`
- Modern LangChain Runnable-based RAG pipeline

---

## Tech Stack

- Python
- LangChain (0.2+)
- Groq LLM (LLaMA 3.1)
- Chroma Vector Database
- HuggingFace Embeddings
- streamlit
- python-dotenv
